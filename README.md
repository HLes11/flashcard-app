# پروژه وب اپلیکیشن فلش کارت (Flashcard Web App)

این یک پروژه ساده دانشگاهی برای ساخت فلش کارت است. این وب اپلیکیشن با استفاده از زبان پایتون (فریم‌ورک Flask) برای بک‌اند و دیتابیس PostgreSQL ساخته شده است. ظاهر برنامه (فرانت‌اند) با HTML، CSS و JavaScript طراحی شده و از تم تاریک (Dark Theme) مشابه Google NotebookLM الهام گرفته است.

## ویژگی‌های پروژه
- **ساخت دسته‌های فلش کارت (Deck):** می‌توانید مانند پوشه‌ها، دسته‌های مختلفی برای فلش کارت‌های خود بسازید (مثلاً: لغات درس اول).
- **ورودی ساده:** کلمات و معانی را به سادگی با فرمت `کلمه : معنی` وارد می‌کنید و برنامه به صورت خودکار آن‌ها را تبدیل به فلش کارت می‌کند.
- **مرور فلش کارت‌ها:** با کلیک روی هر کارت، کارت می‌چرخد و معنی آن نمایش داده می‌شود.
- **طراحی زیبا و ساده:** رابط کاربری تاریک و مینیمال.

## پیش‌نیازها
برای اجرای این پروژه روی سیستم خود، باید موارد زیر را نصب داشته باشید:
1. **Python** (نسخه 3.7 به بالا)
2. **PostgreSQL** (دیتابیس)

## راهنمای نصب و اجرا

### ۱. کلون کردن پروژه
ابتدا پروژه را از گیت‌هاب دانلود یا کلون کنید:
```bash
git clone <لینک-مخزن-گیت‌هاب-شما>
cd flashcard
```

### ۲. ساخت محیط مجازی (Virtual Environment) و نصب کتابخانه‌ها
پیشنهاد می‌شود از محیط مجازی استفاده کنید:
```bash
python -m venv venv
```
فعال‌سازی محیط مجازی در ویندوز:
```bash
venv\Scripts\activate
```
سپس کتابخانه‌های مورد نیاز را نصب کنید:
```bash
pip install -r requirements.txt
```

### ۳. تنظیمات دیتابیس (PostgreSQL)
ابتدا در PostgreSQL یک دیتابیس جدید به نام `flashcard_db` بسازید.
سپس فایل `.env` را در پوشه اصلی پروژه باز کنید و اطلاعات دیتابیس خود (رمز عبور و نام کاربری) را در آن وارد کنید:
```env
DB_HOST=localhost
DB_NAME=flashcard_db
DB_USER=postgres
DB_PASSWORD=رمز_عبور_شما
DB_PORT=5432
```

### ۴. ساخت جداول دیتابیس
برای ساخت جداول مورد نیاز در دیتابیس، فایل `db.py` را یک بار اجرا کنید:
```bash
python db.py
```
اگر پیام `Database initialized successfully!` را دیدید، یعنی جداول با موفقیت ساخته شده‌اند.

### ۵. اجرای برنامه
حالا می‌توانید سرور Flask را اجرا کنید:
```bash
python app.py
```
مرورگر خود را باز کنید و به آدرس `http://127.0.0.1:5000` بروید.

## ساختار فایل‌ها
- `app.py`: فایل اصلی بک‌اند (مسیرها و منطق برنامه).
- `db.py`: کدهای مربوط به اتصال به دیتابیس و ساخت جداول.
- `schema.sql`: کدهای SQL برای ساخت جداول دیتابیس.
- `templates/`: پوشه حاوی فایل‌های HTML (ظاهر صفحات).
- `static/`: پوشه حاوی فایل‌های CSS (استایل‌ها) و JS (کدهای تعاملی مثل چرخش کارت).
- `requirements.txt`: لیست کتابخانه‌های پایتون مورد نیاز.

## نحوه استفاده
1. در صفحه اصلی روی دکمه **"+ ساخت فلش کارت جدید"** کلیک کنید.
2. یک عنوان برای دسته فلش کارت خود بنویسید.
3. در بخش محتوا، کلمات خود را با فرمت `کلمه : معنی` وارد کنید (هر کلمه در یک خط جدید).
4. روی ذخیره کلیک کنید.
5. حالا در صفحه اصلی می‌توانید دسته ساخته شده را ببینید و با کلیک روی **"مرور کارت‌ها"** آن‌ها را مطالعه کنید.
